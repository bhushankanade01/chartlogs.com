I have a trading journal app (ChartLogs) built with React + Express + PostgreSQL + Drizzle ORM.

---

## 1. CSV Trade Import
- Add a "Import CSV" button on the Trades page

- Auto-detect and map columns from common broker exports: MT4/MT5, cTrader, Binance
- Show a preview table before confirming import
- Map CSV columns to our trade schema: symbol, entryPrice, exitPrice, lotSize, direction (buy/sell), entryDate, exitDate, profit/loss

- On confirm, bulk insert trades into the database

## 2. Password Security Fix
- Replace the current SHA256 password hashing with bcrypt (use bcryptjs package)
- Salt rounds: 12
- Update the register and login routes to use bcrypt.hash() and bcrypt.compare()

---

## 3. Password Reset Flow
- Add "Forgot Password?" link on the login page

- Send a reset email using Nodemailer with a reset link
- Create a /reset-password/:token page where user sets a new password
- Validate token expiry and mark it as used after reset
