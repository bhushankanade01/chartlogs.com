=== SEO IMPROVEMENTS ===

1. META TAGS
   Update all pages with:
   - Unique, descriptive <title> (currently says "ChartLogs")
   - Meta descriptions (160 chars max)
   - Open Graph tags for social sharing (og:title, og:description, og:image)
   - Twitter Card tags
   - Canonical URLs

   Examples:
   - Home: "ChartLogs - AI-Powered Trading Journal for MT4/MT5"
   - Dashboard: "Your Trading Dashboard | ChartLogs"
   - Analytics: "Trade Analytics & Performance Reports | ChartLogs"

2. STRUCTURED DATA (JSON-LD)
   - Add schema.org markup:
     * Organization (for homepage)
     * SoftwareApplication (for the app)
     * LocalBusiness (if applicable)
     * Pricing schema (for subscription tiers)
   - Schema helps Google understand your site better

3. SITEMAP & ROBOTS.TXT
   - Generate XML sitemap with all key pages
   - Ensure robots.txt allows indexing of public pages
   - Disallow /admin, /dashboard, etc.

4. READABILITY & KEYWORDS
   - Ensure H1, H2, H3 hierarchy is correct
   - Integrate keywords naturally:
     "trading journal", "MT4 sync", "MT5 sync", "forex journal", 
     "trade analyzer", "AI trading reports"
   - Alt text on all images
   - Internal linking between related pages

5. MOBILE OPTIMIZATION
   - Ensure all pages are mobile-responsive
   - Test on various screen sizes
   - Fast load times (optimize images, code splitting)

6. PERFORMANCE
   - Lighthouse score target: 90+

   - Lazy load images below the fold
   - Minify CSS/JavaScript

=== DESIGN SYSTEM (CONSISTENCY) ===

Colors:
- Primary: #4F46E5 (Blue, for CTAs)
- Success: #10B981 (Green, for profits)
- Danger: #EF4444 (Red, for losses)
- Neutral: #6B7280 (Gray, for text)
- Dark mode bg: #0F172A
- Light mode bg: #FFFFFF

Typography:
- Headings: Clean sans-serif (keep current if it's good)
- Body: Sans-serif, 16px base size
- Monospace for code/numbers (trading stats)

Spacing:
- Use 4px/8px/16px/24px/32px grid consistently
- Consistent padding on cards (16px or 24px)

Animations:
- Page transitions: 0.3s ease-in-out
- Hover effects: 0.2s
- Loading spinners: smooth rotation
- Fade-ins: 0.5s staggered
